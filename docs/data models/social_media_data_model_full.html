Social Media Data Model - Redis Key-Value Store

Main Entities
post:<id> (Hash)
  • text
  • timestamp
  • platform
  • country
  • likes, retweets
  • sentiment
  • user_id
  • engagement
# import.py:111-122 r.hset(f"post:{post_id}", mapping={ "text": text[:500], "timestamp": timestamp, "platform": platform, "country": country, "likes": likes, "retweets": retweets, "sentiment": sentiment, "user_id": user_id, "engagement": engagement })
▼
post:<id>:hashtags (Set)
  • { travel, adventure, ... }
# import.py:134 for tag in hashtags: r.sadd(f"post:{post_id}:hashtags", tag)
post:all (Set)
  • { post_id1, post_id2, ... }
# import.py:158 r.sadd("post:all", post_id)
User Entity
social:user:<id> (Hash)
  • user_id
  • post_count
# import.py:183-186 r.hset(f"social:user:{user_id}", mapping={ "user_id": user_id, "post_count": post_count }) r.sadd("social:user:all", user_id)
▼
social:user:<id>:posts (List)
  • [ post_id1, post_id2, ... ]
# import.py:141 r.lpush( f"social:user:{user_id}:posts", post_id )
social:user:all (Set)
  • All user IDs
# import.py:187 r.sadd("social:user:all", user_id)
Secondary Indexes
Index Keys (Set)
platform:twitter:posts
sentiment:joy:posts
hashtag:travel:posts
country:DE:posts
# import.py:135, 147-149 # Hashtag index r.sadd(f"hashtag:{tag}:posts", post_id) # Platform, Country, Sentiment indexes r.sadd(f"platform:{platform}:posts", post_id) r.sadd(f"country:{country}:posts", post_id) r.sadd(f"sentiment:{sentiment}:posts", post_id)
Pattern:
<attribute>:<value>:posts
→ Contains Set of Post-IDs
→ Enables O(1) lookups
Query Example:
# queries.py
posts = r.smembers("platform:twitter:posts")
posts = r.sinter(
    "sentiment:joy:posts",
    "country:DE:posts"
)
Rankings
post:trending (Sorted Set)
  • Member: Post-ID
  • Score: Engagement
# import.py:153-155 if engagement > 0: r.zadd("post:trending", {post_id: engagement})
hashtag:trending (Sorted Set)
  • Member: Hashtag
  • Score: Count
# import.py:179-180 for tag, freq in hashtag_counts.items(): r.zadd("hashtag:trending", {tag: freq})
Query:
# queries.py - Top 10 trending
r.zrevrange("post:trending", 0, 9,
    withscores=True)
r.zrevrange("hashtag:trending", 0, 9,
    withscores=True)
→ O(log N) complexity
